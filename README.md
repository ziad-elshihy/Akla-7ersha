# 🧪 Stack Demo Testing Project

## 📌 About The Project
This project involves testing the Stack Demo Website, a demo e-commerce platform used to simulate real-world user interactions such as login, browsing products, adding items to cart, and checkout.

The goal is to ensure high quality, usability, and reliability of the application through structured testing.

---

## 🚀 Project Highlights
- ✔️ Designed & executed test scenarios for full user journey  
- ✔️ Covered end-to-end workflows  
- ✔️ Applied multiple testing techniques  
- ✔️ Identified and reported defects  
- ✔️ Ensured smooth UI/UX experience
